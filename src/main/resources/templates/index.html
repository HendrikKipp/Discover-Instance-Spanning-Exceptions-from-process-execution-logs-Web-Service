Discover Instance Spanning Exceptions

This web service enables the discovery of instance spanning exceptions (ISE) based on process event logs. ISEs are exceptions that can span multiple process instances of both its own and other processes. There are five different classes of ISEs: Wait, Cancel, Redo, Change and Rework. Further details can be found in the following paper (Link). In addition, the implementation of the web service can be found in the following GitHub Repository (Link).

To use the web service, the process event logs to be examined must first be uploaded in ".xes" format and a few calculation parameters selected. Afterward, the discovered ISEs are displayed together with appropriate Process Performance Indicators (PPI).
